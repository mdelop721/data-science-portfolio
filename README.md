# Portafolio de Data Science (mdelop)

Este repositorio contiene mi portafolio profesional, donde documento mis proyectos de Ciencia de Datos y Machine Learning. El objetivo es demostrar mis habilidades en análisis de datos, modelado predictivo y comunicación técnica.

🌐 **Sitio Web Vivo**: [https://mdelop.github.io/data-science-portfolio/web/](https://mdelop.github.io/data-science-portfolio/web/)

## 📁 Estructura del Repositorio

- `web/`: Archivos del portafolio web (HTML/CSS). Esta es la "cara" pública del proyecto.
- `notebooks/`: Cuadernos de Jupyter con análisis exploratorios y modelos.
- `data/`: Conjuntos de datos utilizados (o enlaces a los mismos).
- `src/`: Scripts auxiliares de Python para procesamiento y utilidades.

## 🚀 Proyectos Principales

### 1. Análisis Inmobiliario
Modelo de regresión para predecir precios de casas utilizando el dataset de Ames, Iowa. Incluye análisis exploratorio y limpieza de datos.
- Notebook: `notebooks/house_prices_analysis.ipynb`

### 2. Dashboard de Emisiones de CO2
Visualización de datos climáticos globales utilizando librerías interactivas para mostrar tendencias por país.
